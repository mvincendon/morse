******************************* Bienvenue ! ***************************************

Pour utiliser le traducteur de morse, il suffit de make 
puis d'exécuter le fichier morse par ./morse.

Ensuite, l'interface utilisateur vous guidera.

Il est possible de changer certains paramètres dans le fichier config.h,
en particulier la durée en millisecondes d'une unité de temps (temps d'un "ti").

Michael Vincendon SARL décline toute responsabilité de l'apparition malencontreuse
d'une "segmentation fault".
